# **App Name**: DeepTerrorGG Portfolio

## Core Features:

- Persistent Navigation Bar: Implement a navigation bar with links to 'Home', 'Artworks', 'About Me', and 'Contact' sections. It remains fixed at the top of the screen.
- Artwork Gallery: Display artwork provided by the user on the 'Artworks' page, ensuring images are presented in a visually appealing way, with optional zoom on click. Limit of 5 artworks for MVP
- Subtle Animations: Incorporate subtle animations on page transitions or when hovering over elements to enhance user experience.

## Style Guidelines:

- Background color: Black (#000000) to provide a dark and immersive feel.
- Text color: White (#FFFFFF) for high contrast and readability against the black background.
- Outline color: Grey (#555555) to subtly define sections and elements.
- Accent: Teal (#008080) for interactive elements like hover states on navigation links.
- Design the website primarily for desktop users, with a focus on larger screen sizes and high-resolution displays.
- Incorporate smooth transitions and subtle hover effects to enhance user engagement without being distracting.
